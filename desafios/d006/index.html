Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Destaque (negrito)
  • Ênfase (italico)
  • Texto marcado
  • Texto inserido
  • Texto deletado
  • Texto sobrescrito
  • Texto subscrito
  • Pré-formatação
  • Fonte monoespaçada (indicando codigo)
  • Citações simples
  • Citações completas
